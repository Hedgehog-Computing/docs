#### `cov(A: a JS array, matrix or tensor, B: a JS array, matrix or tensor)`

**param** `A` a JS array, matrix or tensor

**param** `B` a JS array, matrix or tensor

**returns** - a number representing the covariance of `A` and `B`

This function simply finds the covariance of the two inputs with the following formula:

<MathJaxContext>
      <MathJax>{"$$cov(A, B) = \\frac{1}{N - 1} \\sum_{i=1}^{N}{(A_i - \\overline{A})\\cdot (B_i - \\overline{B})}$$"}</MathJax>
</MathJaxContext>
