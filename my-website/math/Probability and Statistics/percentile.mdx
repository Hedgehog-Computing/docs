#### `percentile(input: a JS array, matrix or tensor, x: number between interval [0, 1])`

**param** `input` a JS array, matrix or tensor

**param** `x` a number between interval [0, 100]

**returns** - an array representing the percentile (all the numbers with index larger than x% of the values)

Input will be flattened to a 1-d JS array.
