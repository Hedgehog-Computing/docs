#### `reshape(input: a JS array, matrix or tensor, sizes: a 1-d array representing the size)`

**param** `input` a JS array, matrix or tensor

**param** `sizes` a 1-d array representing the size

**returns** - a JS array, matrix or tensor with the corresponding size

size should be like [2, 2, 2] or [2, -1]. Here, -1 means automatically matching.
