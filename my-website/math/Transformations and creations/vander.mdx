#### `vander(input: a JS array, matrix or tensor, dim : a number)`

**param** `input` a JS array, matrix or tensor (will be flattened to a 1-d array)

**param** `dim` a number which should be >= 0

**returns** - a vander matrix

A vandier matrix is shown as follows
