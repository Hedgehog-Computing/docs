#### `is_local_min(input: a JS array, matrix or tensor, dim: 0 or 1)`


**param** `input` a JS array, matrix or tensor

**param** `dim` 0 (row) or 1 (column)

**returns** - a matrix or tensor with 1 denoting local min value and 0 denoting not local min

Notice if `dim` is 1, then `input` must be a 2-d matrix

This function simply finds all the local minimum value of the input.
