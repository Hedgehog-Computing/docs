#### `is_number(input: any)`

**param** `input` - Anything to test if it is a number or not

**returns: boolean** - True if it is a 'number' false otherwise. Infinity and NaN are defined to **not** be numbers in Hedgehog Lab. You can use hex and binary inputs as numbers - 0xfff0 etc. 

This function tests if the input is a number or not. It does not allow strings, infinity, NaN. It checks type too.
