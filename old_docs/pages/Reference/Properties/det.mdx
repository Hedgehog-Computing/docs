#### `det(input: any[][])`

**param** `input` A matrix, should be square.

**returns: number** - A number which represents the determinant of the `input`. One useful tool of the determinant is determining if a matrix is invertible: if det is not zero then it is invertible. 

This function simply returns the determinant of a square matrix.
