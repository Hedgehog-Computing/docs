#### all()

**param** none

**returns** none

This function is useful if you don't want to handpick what functions to import. It imports all math functions (including all discussed in this reference) into the lab.



#### test()

**param** none

**returns** none

This function, similar to `all()` will import all the test functions and **run them**. For each function there is a test function named `{functionname}_test.hhs` that throws an exception if it fails. If it passes nothing happens.
