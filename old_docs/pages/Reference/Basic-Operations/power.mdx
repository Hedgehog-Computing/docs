#### `power(base: any[] | Mat | Tensor | number, pow: number)`

**param** `base` the structure or number to take to the power of `pow`. Can be a number, tensor, javascript array, or Mat

**param** `pow` the number of times `base` gets multiplied by itself, i.e. the power. Strictly has to be a number. 

**returns: any[] | Mat | Tensor | number** - In the case of a number, returns the powered number. In the case of a generalized matrix structure, returns the resulting matrix where elements are each taken to the power of `pow`. i.e. **element-wise power**. For multiplying matrices by themselves, consider writing a for loop.
