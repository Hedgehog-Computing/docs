## Latex Support

{"\\(\A+B\\)"}

This book contains/uses better-react-mathjax which can be installed via<br />
``$ npm install better-react-mathjax``
or <br />
``$ yarn add better-react-mathjax``

But before using latex make sure to wrap the content in the following way:


&lt;MathJaxContext&gt;
    &lt;MathJax&gt;
        ...content
    &lt;/MathJax&gt;
&lt;/MathJaxContext&gt;
